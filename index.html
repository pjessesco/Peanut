template<typename E>
struct MatrixExpr
#include <matrix.h>

Base class of every matrix expressions in Peanut.

Peanut implements matrix expression by following CRTP concept to achieve a static polymorphism. Every derived class should follow its interface even they are not forced by the compiler (i.e., pure virtual method). Also they MUST provide their data type via using Type = XXX, and implement eval() method for matrix evaluation. See Matrix<T, Row, Col>::eval() for more explanation.

Template Parameters

E – Derived class. MatrixExpr
